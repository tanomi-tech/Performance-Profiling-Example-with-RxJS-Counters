# Performance Profling Example with RxJS Counters

## Requirements
- `pnpm`, `yarn` or `npm` :: `pnpm` is recommended as it was used originally and has generated the lockfile

## Setup
Install dependencies.
```sh
pnpm install
# or npm install
# or yarn 
```

Start dev server.
```sh
pnpm dev
# or npm run dev
# or yarn dev
```
